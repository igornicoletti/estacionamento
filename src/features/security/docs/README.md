# Security

Feature responsável por resumo de segurança da conta autenticada.

## Estrutura

- `constants/`: textos e labels.
- `types/`: contratos da feature.
- `services/`: integração com sessão/autenticação.
- `hooks/`: composição de estado.
- `components/`: blocos de UI da feature.
- `routes/`: entrypoint da rota.
